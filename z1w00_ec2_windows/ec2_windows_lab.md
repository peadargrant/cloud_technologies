Prerequisities
==============

This week’s lab assumes that you:

-   Can use the AWS CLI (by getting credentials from AWS Educate for it)

-   Can run Powershell scripts to setup your environment

-   Have deleted `LAB_VPC` if it exists.

-   Have gone through the Linux lab from last week (as we will mainly
    focus on the differences)

Lab task
========

Instance setup
--------------

Set up a Windows EC2 instance following the steps in the lecture notes.

Confirm that you can connect using RDP.

SSH server setup
----------------

Set up the SSH server following the instructions in the lecture notes.
