LAB
===

S3 buckets

Have the ability to host a website, but can do a lot more!
- Simply allow GET to the whole internet.

Names: globally unique across AWS.  Think about yours carefully!

Permissions: very fine grained.
POLICIES LIKE:
{
   "Version":"2012-10-17",
   "Statement":[{
 	"Sid":"PublicReadForGetBucketObjects",
         "Effect":"Allow",
 	  "Principal": "*",
       "Action":["s3:GetObject"],
       "Resource":["arn:aws:s3:::example-bucket/*"
       ]
     }
   ]
 }
