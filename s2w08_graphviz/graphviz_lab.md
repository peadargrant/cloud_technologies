---
title: GraphViz lab
---

1.  Get access to GraphViz, ideally locally by installing it on your own
    system. If (and only if) that really isn’t an option then try the
    web version.

2.  Work through the walkthrough example from scratch. Make sure that
    you can type out the file and can compile it into a PNG image.

3.  Try to diagram out the key components of your project using
    GraphViz. Start off with the basic structure. Use subgraph to group
    related nodes together. Where relevant, place labels onto edges.

4.  Experiment with the styling options to improve the clarity of
    your diagram.
