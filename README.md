# Cloud computing content

This repository holds common content for cloud architecture, cloud foundations and cloud technologies.
This is due to the transitional academic year 2021/22 where formerly Y4 content moves to Y3.

## Cloning this repository

Although the lab content is identical, the CA content is not.
**Make sure** that you **clone the correct branch** of this repository depending on which module you are taking.

## Y4 Cloud Architecture

To get the cloud architecture version of this module, clone:

	git checkout -b architecture https://github.com/peadargrant/cloud

## Y3 Cloud Technologies

To get the cloud techologies version of this module, clone: 

	git checkout -b technologies https://github.com/peadargrant/cloud

## Y3 Cloud Foundations

To get the cloud foundations version of this module, clone:

	git checkout -b foundations https://github.com/peadargrant/cloud
